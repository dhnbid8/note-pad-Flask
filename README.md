# note-pad-Flask
a simple web note pad 

install and run : 

``` 
git clone https://github.com/dhnbid8/note-pad-Flask.git
``` 

```
cd note-pad-Flask
```

```
virtualenv .env
```

```
source .env/bin/activate
```
<br>

```
pip install requirements.txt
```

```
python app.py
```
OR

```
./app.py
```
